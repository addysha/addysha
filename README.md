> Python · React · AWS · Redis · PostgreSQL · MQTT · CAN Bus

- **600+ messages/min** sustained during live track runs, **10 Hz** dashboard refresh
- **<250ms** fault detection across 150+ signals (BMS, MCU, VCU) via Redis pre-fault engine
